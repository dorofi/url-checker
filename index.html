üöÄ What is URL Checker?

URL Checker is a high-performance, asynchronous tool written in Rust that allows you to monitor the status of hundreds of URLs simultaneously. Whether you're a developer checking API endpoints, a DevOps engineer monitoring service health, or a website owner tracking uptime, URL Checker provides professional-grade monitoring with beautiful output and comprehensive reporting.

Note: This webpage (index.html) is an informational landing page that explains the project. The actual URL checking functionality is available through:

  • üñ•Ô∏è Command Line Interface (CLI) - Run from terminal
  • üñ•Ô∏è Desktop GUI Application - Beautiful graphical interface (see Desktop GUI section)

Why doesn't this webpage work as an app? This is just a static HTML page for documentation. The desktop GUI application uses Tauri framework which requires a Rust backend - it cannot run in a regular browser. To use the GUI, you need to run the desktop application (see Desktop GUI section below).

üì¶ Installation

Command Line Tool (CLI)

Prerequisites

  • Rust 1.70 or later
  • Internet connection

Quick Start

# Clone the repository
git clone https://github.com/dorofi/url-checker.git
cd url-checker

# Build the project
cargo build --release

# Run it!
cargo run --release

Desktop GUI Application

Prerequisites

  • Rust 1.70 or later
  • Node.js v16 or later
  • System dependencies (see Desktop GUI section below)

Installation

# 1. Install system dependencies (Linux)
sudo apt-get update && sudo apt-get install -y \
    libwebkit2gtk-4.0-dev build-essential \
    libssl-dev libgtk-3-dev \
    libayatana-appindicator3-dev librsvg2-dev pkg-config

# 2. Install Tauri CLI
cargo install tauri-cli

# 3. Navigate to GUI directory
cd gui

# 4. Install dependencies
npm install

# 5. Run the desktop app
npm run tauri:dev

üíª Usage Examples

Command Line (CLI)

# Basic usage
cargo run --release

# High concurrency (50 simultaneous requests)
cargo run --release -- -c 50

# Custom timeout (30 seconds)
cargo run --release -- -t 30

# Custom input/output files
cargo run --release -- -i my-urls.txt -o my-report.csv

# Export to JSON
cargo run --release -- -i urls.txt -o report.json --format json

Desktop GUI Application

Simply run npm run tauri:dev in the gui/ directory. A desktop window will open where you can:

  • Enter URLs directly in the text area
  • Configure concurrency and timeout settings
  • Click "Check URLs" to start monitoring
  • View real-time results and statistics

URL Checker includes a beautiful desktop application built with Tauri! Get a modern graphical interface with real-time updates, visual statistics, and an intuitive design.

What is Tauri?

Tauri is a framework that combines web technologies (HTML, CSS, JavaScript) with a Rust backend to create lightweight, secure desktop applications. The GUI uses web technologies for the interface, while Rust handles the heavy lifting of URL checking.

How to Run the Desktop App

# 1. Install system dependencies (Linux)
sudo apt-get update && sudo apt-get install -y \
    libwebkit2gtk-4.0-dev build-essential \
    libssl-dev libgtk-3-dev \
    libayatana-appindicator3-dev librsvg2-dev pkg-config

# 2. Install Tauri CLI
cargo install tauri-cli

# 3. Navigate to GUI directory
cd gui

# 4. Install Node.js dependencies
npm install

# 5. Run the application
npm run tauri:dev

Note: The first build takes 5-10 minutes as Rust compiles dependencies. Subsequent builds are much faster!
